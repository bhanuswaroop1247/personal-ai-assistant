## My Research Philosophy

Good research is not about publishing — it's about producing knowledge that is reproducible, interpretable, and honestly reported. I want every paper I contribute to to be something I can defend under rigorous scrutiny.

---

## Core Principles I Work By

### 1. Reproducibility First
- All experiments tracked with random seeds, version numbers, and environment specs
- Results should be reproducible from the same code and data
- Code committed to Git with meaningful commit messages
- Raw data is never modified; processing always applied in scripts

### 2. Honest Reporting
- Report all metrics, not just favorable ones
- Acknowledge limitations clearly in the Discussion section
- Do not cherry-pick hyperparameters after seeing test set results
- OOB error methodology chosen specifically to reduce risk of overfitting to validation

### 3. Calibrated Confidence
- Distinguish between what the data shows and what I interpret
- Use confidence intervals where possible, not just point estimates
- Acknowledge when sample size limits generalizability

---

## Academic Workflow

### Paper Reading Protocol
1. Read abstract + conclusion first to assess relevance
2. Examine methodology for validity (correct use of train/test, appropriate metrics)
3. Note key claims and check if data supports them
4. Extract: method, dataset, results, limitations → into Zotero

### Writing Protocol
1. Outline before writing (section headings + 1-sentence summary each)
2. Write Methods first (most technical, least creative — do it when fresh)
3. Results section: data first, interpretation second
4. Discussion: connect results to literature, acknowledge limitations honestly

## Literature Management

- **Tool:** Zotero with Better BibTeX plugin
- **Library:** ~120 papers organized by topic (KCN, RF methods, pupil tracking)
- **Annotation habit:** Highlight key claims + add notes in PDF reader
- **Citation style:** APA 7 for most journals, Vancouver for medical journals
